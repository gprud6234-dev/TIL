### Context
n8n을 로컬 환경이 아닌 클라우드에서 24시간 운영하고 싶지만, 유료 서버 비용이 부담스러웠습니다. 이를 해결하기 위해 GCP의 Always Free 트라이얼을 활용하여 100% 무료로 n8n을 클라우드에 호스팅하기로 했습니다.

### Core
* **GCP e2-micro**: 구글 클라우드의 'Always Free' 트라이얼을 사용하여 기본 서버 자원을 확보했습니다.
* **Docker**: `docker-compose`를 이용하여 n8n 환경을 컨테이너로 격리하여 설치 및 관리의 효율성을 높였습니다.
  ```yaml
  version: '3'
  services:
    n8n:
      image: n8nio/n8n
      ports:
        - "5678:5678"
  ```
* **Ngrok**: 고정 IP나 복잡한 방화벽 또는 도메인 설정 없이 Ngrok을 이용하여 외부 웹훅(Webhook) 경로를 안전하게 노출시켰습니다.
  ```bash
  ngrok http 5678
  ```

### Insight
유료 클라우드 서비스가 왜 비싼지를 깨달을 수 있었고, 100% 무료로 설정하는 과정에서 여러 복잡한 문제에 부딪혔습니다. 특히, 도메인 주소를 DuckDNS로 설정하고 Let's Encrypt로 인증서를 발급하는 과정에서 CAA 오류가 발생했지만, Ngrok을 사용하여 이를 해결할 수 있었습니다. 이로써 비용 부담 없이 완전 무료로 시스템을 구축할 수 있었습니다.
