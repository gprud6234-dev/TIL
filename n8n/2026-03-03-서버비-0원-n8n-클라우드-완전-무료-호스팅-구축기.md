### Context
n8n을 로컬 환경이 아닌 클라우드에서 24시간 운영하고 싶지만, 유료 서버 비용이 부담스러웠습니다. 그래서 Google Cloud Platform(GCP)의 Always Free Tier를 활용하여 100% 무료 인프라를 구축하는 방법을 시도했습니다.

### Core
* **GCP e2-micro**: Google Cloud의 Always Free Tier를 사용하여 기본 서버 자원을 확보했습니다. 이는 작은 규모의 워크로드에 적합한 무료 인스턴스입니다.
* **Docker**: `docker-compose`를 활용하여 n8n 환경을 컨테이너로 격리하여 설치 및 관리 효율성을 높였습니다.
* **Ngrok**: 고정 IP나 복잡한 방화벽 및 도메인 설정 없이 Ngrok 터널링을 통해 외부 웹훅(Webhook) 경로를 안전하게 노출시켰습니다.

설정 예시는 다음과 같습니다:
```dockerfile
docker-compose.yml
version: '3'
services:
  n8n:
    image: n8nio/n8n
    ports:
      - "5678:5678"
    environment:
      - N8N_BASIC_AUTH_ACTIVE=true
      - N8N_BASIC_AUTH_USER=user
      - N8N_BASIC_AUTH_PASSWORD=password
```

### Insight
유료 클라우드의 비용이 왜 높은지 알게 되었습니다. 완전 무료로 설정하려면 많은 설정 작업이 필요했습니다. 도메인 주소 설정 시 DuckDNS 도메인과 Let's Encrypt 인증서 발급 중 CAA 오류가 발생하여, 해결하는 데 3시간 넘는 시간이 소요되었습니다. 결국 Ngrok을 대체 이용하여 문제를 해결했지만, 완전 무료 조합을 찾아내서 다행히 잘 마무리할 수 있었습니다.
