### Context
n8n을 로컬에서 실행하지 않고 클라우드 환경에서 24시간 운영하고자 했습니다. 그러나 유료 서버비는 부담스러워서 GCP의 Always Free 프리트라이얼을 활용해 100% 무료 인프라를 구축하였습니다.

### Core
* **GCP e2-micro**: Google Cloud Platform의 프리트라이얼 인스턴스를 사용하여 기본 서버 자원을 확보했습니다.
* **Docker**: `docker-compose`를 통해 n8n 환경을 컨테이너로 격리하여 설치 및 관리 효율을 높였습니다.
* **Ngrok**: 고정 IP나 복잡한 방화벽/도메인 설정 없이 Ngrok 터널링을 사용하여 외부 웹훅(Webhook) 경로를 안전하게 노출했습니다.

```yaml
docker-compose.yml 예시:
version: '3'
services:
  n8n:
    image: n8nio/n8n
    restart: always
    ports:
      - 5678:5678
    environment:
      - N8N_BASIC_AUTH_ACTIVE=true
      - N8N_BASIC_AUTH_USER=<사용자명>
      - N8N_BASIC_AUTH_PASSWORD=<비밀번호>
```

### Insight
유료 클라우드 서비스가 왜 비싼지 이해하게 되었습니다. 100% 무료로 설정하려니 설정이 상당히 까다롭습니다. 도메인 주소 설정 시 DuckDNS 도메인을 통해 Let's Encrypt 인증서를 발급하려 했으나 CAA 오류로 어려움을 겪었습니다. 하지만 Ngrok으로 대체하여 문제를 해결했습니다. 모든 설정을 무료로 할 수 있어서 다행입니다!
