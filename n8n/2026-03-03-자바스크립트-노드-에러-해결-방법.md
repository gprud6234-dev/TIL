### Context
GitHub에서 가져온 바이너리 파일을 텍스트로 변환하는 n8n의 '자바스크립트(Code)' 노드에서 실행 에러가 발생했습니다. 에러 메시지는 `ReferenceError: readmeText is not defined`로, 데이터 변환 과정에서 변수명이 일치하지 않아 문제가 생겼습니다.

### Core

```javascript
// 수정 전

// 수정 후
const readmeText = await ... // 2번째 줄

const content = readmeText.toString('utf-8'); // 5번째 줄
```

### Insight
코드에서 변수를 선언하고 사용할 때는 변수명이 정확히 일치하는지 확인하는 것이 중요합니다. `not defined` 에러가 발생하면, 먼저 변수명이 올바르게 사용되었는지 검토하세요. 변수명 오타는 코드 실행의 주된 오류 원인 중 하나입니다.
