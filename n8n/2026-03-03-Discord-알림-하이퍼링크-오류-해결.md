### Context
n8n 워크플로에서 Discord로 전송하는 알림 메시지에 포함된 하이퍼링크가 제대로 작동하지 않는 문제가 발생했습니다. 마크다운 링크 문법 오류(닫는 소괄호 누락)를 수정했으나 여전히 링크와 미리보기(Embed) 카드가 일반 텍스트로만 표시되는 현상이 반복되었습니다.

### Core
문제를 해결하는 과정에서 URL 주소 중간에 불필요한 대괄호([) 기호가 남아있는 것을 발견했습니다. 이로 인해 Discord가 링크를 올바르게 파싱하지 못하고 있었습니다.

오류 구문:
```
.../blob/main/[n8n/2026-03-03...
```

적용된 정상 코드:
```markdown
[👉🏻 기록된 TIL 바로보기](https://github.com/gprud6234-dev/TIL/blob/main/{{ $('AI Agent').item.json.output.folder }}/{{ $('AI Agent').item.json.output.filename }})
```

이 코드를 전체적으로 덮어쓰는 방식으로 문제를 해결했습니다.

### Insight
링크나 마크다운 문법에서 오류가 계속 발생한다면, 보이지 않는 '숨은 불청객 기호(오타)'가 남아 있을 가능성이 큽니다. 부분적인 수정으로는 오류가 계속될 수 있으므로, 처음부터 문법 구조가 명확한 코드로 전체를 덮어쓰는 것이 보다 빠르고 정확한 해결 방법일 수 있습니다.
