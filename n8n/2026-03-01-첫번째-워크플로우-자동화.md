### Context
n8n을 사용하여 디스코드 메시지를 수신할 때 자동으로 트리거가 발생하는 워크플로우를 구현해보았습니다. 이 워크플로우는 메시지를 수신하면 TIL 형식으로 변환된 내용을 GitHub에 push하고, 완료 후 README.md 파일에 인덱스를 추가하며 최종 완료 메시지를 발신합니다.

### Core
```yaml
- Trigger: Discord 메시지 수신
- Action 1: 메시지를 TIL 형식으로 변환
- Action 2: 변환된 내용을 GitHub에 push
- Action 3: README.md 파일 업데이트
- Action 4: 완료 메시지 발신
```

### Insight
n8n을 처음 사용해보았는데, 기본적인 노드의 input과 output만 이해한다면 워크플로우를 만들기 어렵지 않습니다. 복잡한 자동화 작업을 쉽게 처리할 수 있음을 실감했습니다. 이 모든 것을 배우도록 도와준 n8n 커뮤니티에 감사를 전합니다.
